## 功能验收

- [ ] 红人档案、当前商品和完整上下文可被千川、价值观、直播脚本读取。
- [ ] 千川仿写：初稿可预审、最多四轮、人工确认后保存和导出。
- [ ] 价值观仿写：旧版四步、锁定开头、商品只参与方向推导。
- [ ] 直播仿写：独立入口保持可用，工作台入口带入红人和当前商品。
- [ ] 素材库：六类资料、文档解析编辑、私有视频上传/播放/替换/删除、红人路径隔离。
- [ ] 复盘：多文件逐份文件名/正文对应、完整红人上下文、失败不伪完成、保存/导出。
- [ ] 千川成片预审：两条完整视频上传、流式报告、保存/导出、异常清理与日志；不得使用关键帧替代。
- [ ] 新版既有工具和额外页签仍可访问；TikTok 工具未纳入本轮。
- [ ] 管理端日志可查到调用用户、功能、模型、状态和耗时；产出中心可查保存结果。
