## 一票否决项

```
1. 有自主注册入口
2. operator 能访问 /admin/* 页面
3. operator 能看到其他用户的任务或产出
4. 前端直连 AI / TikHub / OSS（F12 Network 中出现外部请求）
5. 密码或密钥在前端明文展示
6. 页面白屏或 JS 报错导致功能不可用
```
